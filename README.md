# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'WrbSsoCert'
Import-Module -Name 'WrbSsoCert'
```

To install privately:

```powershell
Save-Module -Name 'WrbSsoCert' -Path '.'
Import-Module -Name '.\WrbSsoCert'
```
